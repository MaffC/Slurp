Slurp by mudtail

File upload and URL shortening service, with user and file management facilities.

Slurp! started out as a simple URL shortener, which then gained file uploading and user registrations. This project is named after the song Slurp! by Desert Planet.

Dependencies

Web server with PHP >= 4.3 MySQLi PHP extension MySQL server Writeable file system

Install

Grab a copy of Slurp! from GitHub. Extract it to your htdocs folder Navigate to yourdomain.com/install Run through the setup Done!
